# Game Vision

## Design sentence

A bullet hell roguelite where danger is not only a threat — danger is a build resource.

## Core fantasy

The player survives unstable portals, cursed trades, boss chaos, and impossible-looking decisions. The best runs should feel partially planned, partially improvised, and partially cursed.

## Target feeling

Players should often think:

> I probably should not do this... but if it works, this run becomes insane.

## Core pillars

### 1. Risk creates power

Dangerous choices should not only punish the player. They should create new build paths.

### 2. Every negative can become useful

A negative modifier should have at least one build that can exploit it.

Examples:

- Faster enemies become useful for thorns, explosions, lifesteal, and kill-chain builds.
- Low HP becomes useful for last-stand builds.
- More bosses become useful for boss-reward builds.
- Curses become useful for corruption-scaling builds.

### 3. Weird builds are intended

The game should support builds that look bad at first but somehow work through strange synergies.

### 4. Portals are the chaos engine

Portals are the main system that creates danger, opportunity, and memorable run stories.

## Art direction

Simple readability like Brotato, combined with dark-cute occult energy inspired by Cult of the Lamb.

The art should support fast combat clarity first. Style polish comes after the prototype proves the gameplay loop.
