# Bullet Hell Roguelite Prototype

A 2D bullet hell roguelite prototype built in Godot.

## Core identity

Danger is not only a threat. Danger is a build resource.

The main hook is a portal system that creates risky events, extreme boss situations, cursed trades, and strange build opportunities. The player should constantly think: "This is probably a bad idea... but my build might actually abuse it."

## Current prototype goal

Build the smallest playable version that proves the portal philosophy:

1. Move around in an arena.
2. Fight simple enemies.
3. Trigger risky portals.
4. Survive chaotic events.
5. Get rewards that can create weird builds.

## Engine

- Godot 4.x
- 2D
- GDScript first

## First milestone

WASD movement in an empty arena with a player foundation and base stat block.
